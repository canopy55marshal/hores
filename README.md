# Trae AI 基因养成赛马 (Trae AI Gene Breeding Horse Racing)

一款基于 Web 的文字交互类赛马游戏，融合了“竞速竞技+基因养成+寿命管理+NBA式选秀”四大核心玩法。

## 📖 核心玩法
- **竞速竞技**：文字模拟赛马过程，结合基因技能触发，体验紧张刺激的比赛。
- **基因养成**：收集与杂交不同基因（疾风、坚韧、爆冲等），培育专属超级赛马。
- **寿命管理**：赛马有寿命限制，需合理规划参赛与传承，退役后可作为种马传承优良基因。
- **NBA式选秀**：赛季结束根据战绩进行选秀，获取稀有赛马，支持选秀权交易。

## 🛠️ 部署说明
本仓库已配置 GitHub Actions 自动部署到 GitHub Pages。

### 快速部署
1. Fork 本仓库。
2. 在仓库 Settings -> Pages 中启用 GitHub Actions 作为构建源。
3. 推送代码到 main 分支即可自动触发部署。

### 本地开发
1. 克隆仓库。
2. 使用 VS Code 或其他编辑器打开。
3. 运行 `index.html` 或 `preview.html` 即可预览。
